# SQL-Business-Analytics-for-Digital-Music-Store
“In this project, I analyzed a digital music store database using SQL to gain insights into sales and customer behavior.
I wrote complex queries using joins, aggregations, subqueries, and window functions to analyze different aspects of the data.
Key insights included identifying top-spending customers, most popular music genres like Rock, and countries generating the highest revenue. 
I also analyzed artist and album performance.
These insights are valuable for building recommendation systems, targeting high-value customers, and focusing on profitable regions, 
ultimately improving customer satisfaction and business growth.”
“This project helped me strengthen my skills in data cleaning, analysis, and deriving actionable insights.”
